Task 4: Page Title and Meta Tags

Below is an example of how we can add Meta Tags in HTML for SEO optimization 


<head>
  <!-- Meta tags for SEO and accessibility -->
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <!-- SEO Meta Tags -->
  <meta name="description" content="This is a sample webpage demonstrating the use of meta tags, SEO optimization, and accessibility features in HTML." />
  <meta name="keywords" content="HTML, meta tags, SEO, accessibility, page title" />
  <meta name="author" content="Hardik Mistry" />
</head>
